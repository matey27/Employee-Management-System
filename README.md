# Employee Management System (EMS)
A centralized platform built with Java, JDBC, and MySQL.

### Key Features:
* **Digitized Records:** Replaced scattered spreadsheets with a structured MySQL database.
* **High Performance:** Optimized retrieval time to **<10 seconds** using SQL Indexing.
* **Efficiency:** Improved HR reporting speed by **50%** through automated reporting scripts[cite: 1, 2].

### Tech Stack:
* Java (JDK 16)
* JDBC (MySQL Connector)
* MySQL Database
